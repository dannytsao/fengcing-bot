- Resolved

## Logging Rules

Record an entry here when:

- A production incident occurs.
- A recurring bug pattern appears.
- A risk affects delivery or reliability.
- A decision creates a known tradeoff.
- A deferred issue should remain visible across phases.

Each entry should link to a GitHub Issue whenever possible.
